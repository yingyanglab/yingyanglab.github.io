# Ying Yang — Academic Homepage

A responsive academic homepage built with plain HTML, CSS, and JavaScript.

## Local preview

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## GitHub Pages

In the repository settings, open **Pages**, choose **Deploy from a branch**, select the `main` branch and `/ (root)`, then save.

Before publishing, replace all bracketed placeholder content in `index.html` with verified information.
